# Simple-board
A simple web board application built with Flask and SQLite.

How to run

1. Clone repository
git clone https://github.com/yourname/Simple-board.git
cd Simple-board

2. Create virtual environment
python -m venv venv
venv\Scripts\activate

3. Install dependencies
pip install -r requirements.txt

4. Initialize database
python init_db.py

5. Run application
python app.py

6. Open browser
http://127.0.0.1:5000
